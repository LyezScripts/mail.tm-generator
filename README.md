# mail.tm bulk generator

A simple Mail.tm bulk account generator. Just run `main.py` and it handles everything automatically from start to finish.

It generates random usernames, creates accounts using the Mail.tm API, and saves any successful accounts locally.

## How to use

Install requirements:
pip install requests colorama


Run the script:
python main.py

Follow prompts on screen

sends to Account.txt, name:password

Run it and it handles everything automatically.
A basic bulk generator for Mail.tm temporary email accounts.
